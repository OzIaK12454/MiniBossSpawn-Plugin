# MiniBossSpawn-Plugin
[Paper/Bukkit] 1.21.x MiniBossSpawn Plugin by OZI09

Version of Plugin: 1.0
Version Minecraft: 1.21.x
Version Tested Server: 1.21.10 Bukkit

Bored with the usual mobs in Minecraft?
With this plugin you can spawn any MiniBoss you want, set it in the config, reload the plugin and use /spawnboss <boss name>
